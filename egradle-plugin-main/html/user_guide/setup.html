Preferences

Go to preferences, search for EGradle settings, setup your root project path and (optional) the default JAVA_HOME to set for gradle wrapper.

Defaults

Insie this area you can define
  • Gradle root project path
  • JAVA_HOME - an optional setup which is available for all your launch configurations
