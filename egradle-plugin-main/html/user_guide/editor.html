Editor

EGradle has an integrated small editor with following features:

  • Syntax highlighting.
    • All JAVA keywords
    • Strings
    • Comments
    • Some Groovy keywords
    • Some Gradle well known closures
    • Some Gradle predefined variables
  • URL linking - press CTRL + mouse click
  • ...new features are planned (e.g. link to included files)
